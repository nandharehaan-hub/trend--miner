## 🚀 Getting Started

### Prerequisites
- Python 3.7 or higher
- Basic knowledge of financial markets (helpful but not required)

### Installation

1. **Clone the repository**:
```bash
git clone https://github.com/nandharehaan-hub/trend--miner.git
cd trend--miner
```

2. **Install dependencies** (if available):
```bash
pip install -r requirements.txt
```

3. **Run the demo**:
```bash
python demo.py
```

The AI follows a systematic approach:
